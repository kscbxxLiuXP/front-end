### 问题

害，这问题就多的去了

写在这里，以后有时间(不会的)就来修复

- 和后端项目异步交互时会偶尔出现莫名的bug
- Header右侧的图标 CSS加载异常，一加其他的内容就会格式错误，显示异常，所以保持现有的这几个就挺好
- 有时候登录验证码输入对的也会提示输入错误
- 退出登录时，浏览器的console会报error
